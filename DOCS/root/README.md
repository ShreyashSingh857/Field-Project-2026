## Setup

1. Clone the repository.
2. Copy backend/.env.example to backend/.env and fill in environment values.
3. From backend, run npm run seed:demo.
4. Start backend on port 5000.
5. Start apps/user-app on port 5173.
6. Start apps/admin-app on port 5174.
7. Start apps/worker-app on port 5175.
